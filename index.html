JavaScript

JavaScript é uma linguagem de programação que permite a você criar conteúdo que se atualiza dinamicamente, controlar múltimídias, imagens animadas, e tudo o mais que há de intessante.

Vejamos aqui um exemplo de código em JavaScript:

                
                    // index.js

                    document.addEventListener('DOMContentLoaded', function() {
                        // Inicializa o contador
                        let contador = 0;

                        // Função para incrementar o contador e atualizar o texto
                        function incrementarContador() {
                            contador++;
                            // Seleciona o elemento do contador pelo ID e atualiza o texto
                            document.getElementById('counter').textContent = 'Contagem: ' + contador;
                        }

                        // Adiciona um ouvinte de eventos ao botão
                        document.getElementById('incrementButton').addEventListener('click', incrementarContador);

                        // Adiciona um console.log para verificar se o evento DOMContentLoaded é acionado
                        console.log('DOMContentLoaded');
                    });

Este código é um código básico do Js (JavaScript), que básicamente adiciona um botão na sua tela e um contador, e toda vez que você clica nele, ele adiciona +1 à conta.

Não esqueça que este código está vinculado à um arquivo HTML.

TypeScript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft que é uma extensão do JavaScript. Ela adiciona recursos avançados ao JavaScript, como a tipagem estática e interfaces, tornando mais fácil detectar e prevenir erros durante a fase de desenvolvimento

Vejamos aqui alguns tópicos do porquê TypeScript é importante:

Tipagem Estática:

TypeScript adiciona um sistema de tipagem estática ao JavaScript, permitindo aos desenvolvedores definir tipos para variáveis, parâmetros de função e retorno de função. Isso ajuda a detectar erros durante o desenvolvimento, antes mesmo de executar o código.

Produtividade Aprimorada:

Com tipagem estática, os editores de código e IDEs (Ambientes de Desenvolvimento Integrado) podem fornecer recursos como autocompletar, refatoração e sugestões de correção de erro em tempo real. Isso aumenta significativamente a produtividade dos desenvolvedores.

Manutenção de Código:

A tipagem estática torna o código mais legível e compreensível, facilitando a manutenção a longo prazo. Ajuda a evitar bugs comuns relacionados a tipos e fornece uma documentação implícita do código.

Orientação a Objetos e Módulos:

TypeScript oferece suporte a recursos de programação orientada a objetos, como interfaces, classes e herança. Além disso, facilita a organização do código por meio de módulos, tornando-o mais modular e fácil de manter.
